本地表格机器学习分析平台

说明书

If you have any suggestions for improvement, please open an issue on GitHub

数据源

导入 CSV/Excel 或示例数据，决定后续分析、建模和预测使用的数据表。

任务设置

选择回归、分类或聚类任务，指定目标列和特征列，会直接影响模型训练和结果解释。

字段角色

导入数据后，勾选哪些列作为特征，哪些列作为目标。目标支持多选。

处理与训练

设置缺失值处理、文本编码、模型和训练参数，用来控制数据清洗方式和模型复杂度。

版本与引用

复制软件引用和 BibTeX，便于在论文、报告或补充材料中注明工具来源。

字段摘要

查看每列的数据类型、缺失、分布和样本值，用于判断哪些字段适合建模。

绘图设置

选择图表类型、坐标列和导出样式，可调整颜色、字体、字号、标签距离和导出格式。
