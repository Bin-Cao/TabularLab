# Version

## V1.1.0 - 2026-07-01

Interface, visualization, and documentation upgrade release.

### Added

- Commercial-style application layout with resizable left sidebar.
- Expanded chart styling controls for colors, fonts, sizes, label spacing, tick spacing, label angles, and chart dimensions.
- PNG and SVG chart export using the same configurable chart style.
- Violin plot, configurable correlation heatmap styles, and adjustable heatmap feature count.
- Task-specific result chart controls for regression, classification, and clustering.
- Confusion matrix number size and number color controls.
- t-SNE clustering projection option.
- Result summary report describing input data, task, target, features, evaluation mode, and effect metrics.
- Detailed beginner-oriented user manual with workflow screenshots from `docs/`.

### Changed

- Updated software metadata, citation, README badges, and release links to 1.1.0 / V1.1.0.
- Improved chart defaults and result chart parameter visibility by task type.
- Improved UI visual hierarchy, table styling, panels, tabs, and result presentation.

## v1.0.0 - 2026-06-29

Initial public version of TabularLab.

### Included

- Browser-based CSV, TSV, TXT, XLSX, and XLS data loading.
- Chinese and English interface.
- Regression, classification, and clustering workflows.
- Missing value handling and categorical encoding.
- Train/test split and cross validation.
- Mainstream sklearn-style tabular models implemented in lightweight browser JavaScript.
- Visualization, prediction, and downloadable analysis outputs.
- Example datasets in `data/`, each with CSV, Excel, and bilingual README documentation.
- GitHub citation metadata and BibTeX citation.
