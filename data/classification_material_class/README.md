# Material Class Classification / 材料类别分类数据

## 中文说明

材料类别多分类数据集，目标列为 `material_class`。可用于测试分类模型识别导电金属、结构陶瓷、聚合物复合材料和工程合金。

推荐任务：分类。推荐目标列：`material_class`。

## English Description

A multiclass materials dataset with `material_class` as the target. It can be used to classify conductive metals, structural ceramics, polymer composites, and engineering alloys.

Recommended task: classification. Recommended target: `material_class`.
