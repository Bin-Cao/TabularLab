# Alloy Strength Regression / 合金强度回归数据

## 中文说明

材料领域回归数据集，用于预测 `yield_strength_mpa`。数据包含合金族、工艺路线、主要元素含量、晶粒尺寸、固溶温度、时效时间和析出相体积分数。

推荐任务：回归。推荐目标列：`yield_strength_mpa`。推荐特征：除目标列外的所有列。

## English Description

A materials regression dataset for predicting `yield_strength_mpa`. Features include alloy family, processing route, major element contents, grain size, solution temperature, aging time, and precipitate fraction.

Recommended task: regression. Recommended target: `yield_strength_mpa`. Recommended features: all columns except the target.
