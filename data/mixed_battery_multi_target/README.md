# Battery Materials Multi-target / 电池材料多目标数据

## 中文说明

包含缺失值、类别特征和多目标回归的材料数据集。默认目标为 `capacity_mah_g` 和 `cycle_retention_pct`，可用于测试多目标预测、缺失值处理和文本编码。

推荐任务：回归。推荐目标列：`capacity_mah_g`, `cycle_retention_pct`。

## English Description

A materials dataset with missing values, categorical features, and multi-target regression. Default targets are `capacity_mah_g` and `cycle_retention_pct`. It is suitable for testing multi-target prediction, imputation, and categorical encoding.

Recommended task: regression. Recommended targets: `capacity_mah_g`, `cycle_retention_pct`.
