# Alloy Family Clustering / 合金族聚类数据

## 中文说明

无监督材料聚类数据集，没有目标列。数据包含密度、模量、热膨胀、导热率、成本、耐蚀评分和加工方式，可用于发现合金族或材料性能分群。

推荐任务：聚类。目标列：无需选择。

## English Description

An unsupervised materials clustering dataset with no target column. It contains density, modulus, thermal expansion, thermal conductivity, cost, corrosion score, and processing route for discovering material families or property clusters.

Recommended task: clustering. Target column: not required.
