# Heat Treatment Window Classification / 热处理窗口分类数据

## 中文说明

二分类材料工艺数据集，目标列为 `heat_treatment_pass`。用于判断钢/合金热处理参数是否落在可接受工艺窗口内。

推荐任务：分类。推荐目标列：`heat_treatment_pass`。

## English Description

A binary process-classification dataset for materials heat treatment. The target `heat_treatment_pass` indicates whether the composition and heat treatment parameters fall within an acceptable processing window.

Recommended task: classification. Recommended target: `heat_treatment_pass`.
